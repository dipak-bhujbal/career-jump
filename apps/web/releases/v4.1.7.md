# v4.1.7

Hotfix release for optional API-array handling after the v4.1.6 deploy.

## Highlights

- Fixes an Applied Jobs crash when the API response omits the `jobs` array.
- Normalizes Applied Jobs route arrays before deriving counts, filters,
  drawer state, and company filter options.
- Hardens related profile and dashboard counts against omitted empty arrays.
- Makes the company picker no-results state safe when registry entries are
  missing from an otherwise successful response.

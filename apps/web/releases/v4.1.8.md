# v4.1.8

Hotfix release for Add Company registry response normalization.

## Highlights

- Fixes Add Company crashes when registry search responses omit `entries` or
  `total`.
- Normalizes registry search responses in the query hook so picker consumers
  always receive a safe `entries` array and numeric `total`.
- Makes the picker result-count header and no-results state use the normalized
  values.

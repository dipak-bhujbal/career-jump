# v4.1.6

Patch release for company-picker discoverability and dashboard analytics
reliability.

## Highlights

- Adds curated Tier 1, Tier 2, and Tier 3 starter rows in the Add company
  picker so users can browse a small sample without exposing the full 1,230
  company registry by default.
- Keeps registry search server-limited while still showing the full protected
  registry count in the picker header.
- Makes application-side dashboard widgets derive counts from the same applied
  job source as the Applied Jobs page.
- Recomputes conversion ratios from raw counts as fractions, preventing
  percent-scaling issues such as `770%` application ratio.
- Removes configured-but-unapplied companies from the Top companies widget so
  it represents actual applications.

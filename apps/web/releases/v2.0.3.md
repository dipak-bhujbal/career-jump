# v2.0.3

Patch release focused on auth-backed identity display and a few small UX
consistency fixes across account and company-management surfaces.

## Highlights

- Uses shared auth display helpers so sidebar, profile, and profile menu show
  the resolved Cognito identity instead of placeholder local profile values.
- Keeps already tracked companies visible in the picker while moving them below
  actionable results.
- Refreshes logs after data-reset operations so the reset flow does not leave
  stale observability state behind.

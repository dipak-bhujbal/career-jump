# v3.0.0

Major release establishing the React app as the source of truth for the next AWS
deployment train and starting full separation from the older `career-jump-aws`
application resources.

## Highlights

- Adds a bundled 1,230-company registry fallback so company search remains
  useful even when the backend registry endpoint is unavailable.
- Adds React-owned backend foundation infrastructure for
  `career-jump-web-poc-state` and `career-jump-web-poc-notifications`.
- Wires the full-stack template so Cognito and notification resources consume
  the React-owned DynamoDB table and SNS topic instead of externally supplied
  old-app resources.
- Moves production AWS API/Cognito config into runtime `aws-config.js`, while
  keeping `VITE_*` values available for local development.
- Prevents deployed CloudFront URLs from entering mock/demo mode through
  missing Cognito config or `?demo=1`.

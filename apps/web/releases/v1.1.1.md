# v1.1.1

Patch release for the production monorepo after the first `v1.1.x` deploy.

## Highlights

- Moves support into Profile and removes the duplicate sidebar support entry.
- Fixes configuration UX around saving tracked companies, custom-company flow, and editable job-title filters.
- Adds admin-aware auth and password reset support for both user pools.
- Improves shared company/job scanning behavior and Workday handling for registry-backed companies.

## Validation

- `npm run validate`
- `npm run aws:build`

## Release context

- Release branch: `release/1.1.1`
- Monorepo version: `1.1.1`

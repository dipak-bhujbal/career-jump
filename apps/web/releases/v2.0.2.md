# v2.0.2

Patch release focused on hardening dashboard and configuration screens against
partially loaded API data.

## Highlights

- Guards dashboard widgets when applied-job rows or configuration payloads are
  missing nested objects during load or partial responses.
- Tightens configuration route access to optional `config` payload fields so the
  page keeps rendering cleanly while data is still resolving.

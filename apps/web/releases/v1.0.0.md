# v1.0.0

Initial production-style release for `career-jump-web`.

## Highlights

- Ships the React/TanStack rebuild as a standalone repo on GitHub.
- Produces a successful production build with route-based code splitting.
- Adds an isolated frontend-only AWS stack template so deployment does not
  touch the older `career-jump` resources already in use.
- Boots the shared AWS auth flag via `public/aws-config.js` so the deployed
  frontend can forward the existing Cognito token to the protected backend.

## Validation

- `npm run build`

## Deploy shape

- CloudFormation stack: `career-jump-web-poc`
- S3 bucket: `cj-web-static-poc-<account-id>`
- CloudFront distribution: isolated from the old frontend stack

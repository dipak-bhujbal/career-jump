# Career Jump AWS v2.2.3

Release date: 2026-04-22

## Summary

This patch preserves configured Apps Script notification parameters during GitLab releases.

## Fixes

- `release-prod` only passes Apps Script SAM parameters when the matching GitLab variables are present.
- Existing NoEcho webhook and shared-secret stack parameters are not cleared by a release job that omits optional notification variables.
- Version labels and cache-busting were advanced to v2.2.3.

## Validation

- `npm run check`
- `node --check public/app.js`
- `sam validate --lint --region us-east-1`
- `sam build`

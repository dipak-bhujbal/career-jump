# v3.0.1 — 2026-04-23

## Summary

Patch release to fix the production frontend CSP so the browser can call the AWS Lambda Function URL from the CloudFront-hosted UI.

## What's Fixed

### Security
- CloudFront Content Security Policy `connect-src` now explicitly allows the deployed `us-east-1` Lambda Function URL host pattern
- Browser API requests from the frontend no longer fail with `Load failed` due to CSP blocking the Lambda Function URL origin

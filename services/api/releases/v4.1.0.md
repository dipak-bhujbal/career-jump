# Career Jump AWS v4.1.0

**Released:** 2026-04-25

## Summary

Minor release. Ships the new registry-company picker UI foundation and adds server-side company search for the registry endpoints so large registries are easier to browse and attach to config rows.

## Changes

### Registry API company search

`GET /api/registry/companies` now accepts an optional `search` query parameter.

- `src/routes.ts` trims and lowercases the incoming search term
- results are filtered by case-insensitive company-name match before the response is sliced to the requested limit
- existing `ats`, `tier`, and `limit` filters still work as before

### Company picker UI styling

`public/styles.css` now includes the styling needed for the company picker experience:

- picker shell, header, controls, and result rows
- ATS badges and tier tags for registry metadata
- compact footer actions and small-button treatment
- shared `.hidden` helper for UI state toggles

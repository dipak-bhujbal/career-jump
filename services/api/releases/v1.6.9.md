# Career Jump v1.6.9

Career Jump AWS POC now keeps scan notifications readable during parallel company runs.

## What's improved

- Replaced stacked scan progress toasts with one updating run toast.
- Added fanout-aware progress counting so AWS scans show completed companies across the whole run instead of each Lambda reporting `1/1`.
- Included the company name in per-company inventory messages when the inventory log only includes company counts.
- Suppressed the noisy post-completion `Failed to fetch` toast from the final background refresh while still logging it to the console.
- Bumped frontend asset URLs so CloudFront serves the updated notification code.

# Career Jump AWS v2.2.2

Release date: 2026-04-22

## Summary

This patch completes the notification wiring for the AWS app.

## Fixes

- GitLab `release-prod` now passes optional `APPS_SCRIPT_WEBHOOK_URL` and `APPS_SCRIPT_SHARED_SECRET` variables into the SAM/CloudFormation deployment.
- Email payloads now include both new jobs and updated jobs in the backward-compatible `jobs` array.
- Email payloads also include separate `newJobs` and `updatedJobs` sections for cleaner templates.
- Updated jobs include previous/current field diffs through `updateChanges` plus a ready-to-render `updateDiffText`.
- `/api/debug/email` now sends one sample new job and one sample updated job so the template can be tested before a real scan.

## Validation

- `npm run check`
- `node --check public/app.js`
- `sam validate --lint --region us-east-1`
- `sam build`

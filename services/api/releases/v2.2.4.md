# Career Jump AWS v2.2.4

Release date: 2026-04-22

## Summary

This patch aligns per-company new-job logs, final run summaries, dashboard cards, and Available Jobs filters.

## Fixes

- Latest-run new jobs are now based on the previous stored inventory diff, not historical seen/email markers.
- Company summary new counts and final run new counts now use the same definition except for same-run duplicate grouping.
- Dashboard `New in latest run` and the Available Jobs `New jobs` filter are refreshed from the same marker keys written during finalization.
- Version labels and cache-busting were advanced to v2.2.4.

- `node --check public/app.js`
- `sam validate --lint --region us-east-1`
- `sam build`

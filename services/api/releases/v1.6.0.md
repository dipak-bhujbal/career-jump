Career Jump 1.6 brings a more polished, more resilient daily workflow.

This release improves scan reliability, makes scan recovery much friendlier when something hangs, and refreshes the interface so the app feels calmer and easier to use throughout the day.

## What’s new

- Scan control is more reliable. Only one scan can run at a time, and you can now abort a stuck scan and start a fresh one without waiting for the old lock to clear itself.
- Scheduled and manual scans now share the same single workspace, so emails, logs, dashboard updates, and inventory stay aligned.
- Company fetches now time out cleanly instead of hanging indefinitely, with stronger failure logging around ATS fetch problems.
- Broken-link cleanup is now part of the normal scan flow, so the available inventory stays cleaner automatically.

## Interface improvements

- The app has been visually polished across dashboard, jobs, action plan, configuration, and logs.
- Empty states now guide you toward the next useful action instead of simply reporting that nothing matched.
- Company filters are easier to use with multi-select dropdowns and cleaner table interactions.
- Configuration is now easier to manage with section-based save and cancel actions, a better company editor, and clearer keyword rule editing.
- Status updates and apply actions now include centered feedback moments that feel more intentional and supportive.

## Theme updates

- Dark theme now includes a cinematic meteor background effect behind the main app content.
- Light theme now includes a softer atmospheric background with gentle motion, while keeping the core interface clean and readable.

## Reliability notes

- This release also removes remaining single-user tenancy complexity from the active runtime path.
- The app continues to use KV as the active runtime store, with the archived D1-based design preserved in the repository for future expansion if needed.
